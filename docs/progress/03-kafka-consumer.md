# 📅 Daily #3 — Подключение Kafka Consumer
Дата: 2025-06-21

## 🎯 Задачи
Подключить Kafka consumer, который:
- подключается к брокеру,
- подписывается на топик `metrics`,
- читает сообщения и выводит их в лог (сначала просто в консоль).

## 📦 Используемые библиотеки

- `github.com/segmentio/kafka-go` — простой в использовании Kafka client.
- `zap` — логгер для вывода.
- `context` — управление отменой consumer по Ctrl+C.
