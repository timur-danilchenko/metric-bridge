# Task 9 — Интеграционные тесты и CI-окружение

## Задачи

Добавить возможность запускать интеграционные тесты с поднятием Kafka и Postgres в CI-подобной среде.

## ✅ Сделано

- Создан файл `docker/docker-compose.ci.yml` для CI окружения
- Добавлены команды в `Makefile`:
  - `test_ci`: запуск окружения, миграций и интеграционных тестов
  - `compose_ci`, `compose_ci_down`
- Интеграционный тест `TestKafkaToPostgresFlow` проверяет:
  - доставку метрики в Kafka
  - обработку и запись в Postgres
- Добавлена автоматическая миграция базы при запуске `test_ci`
- Добавлено ожидание готовности сервисов и worker-процесса

## 🧪 Как протестировать

```bash
make test_ci
