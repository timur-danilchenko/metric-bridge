# MetricBridge

**MetricBridge** — это высокопроизводительная система доставки, обработки и хранения метрик в реальном времени, построенная на Golang.

Проект демонстрирует архитектурный подход к созданию микросервисов с использованием Kafka, PostgreSQL, Prometheus и параллельной обработки данных через goroutines.



## 📦 Основной стек технологий

| Компонент     | Назначение |
|---------------|------------|
| **Golang**    | Основной язык реализации |
| **Kafka**     | Очередь сообщений для доставки метрик |
| **PostgreSQL**| Хранилище метрик |
| **Prometheus**| Мониторинг и экспорт внутренних метрик сервиса |
| **goroutines**| Параллельная обработка метрик |


## 🧩 Архитектура (MVP)
```plaintext
            [ Клиенты ]
                 ↓
      [ Kafka (топик метрик) ]
                 ↓
 [ Go worker ] (читает, обрабатывает)
       ↓                    ↓
 [ PostgreSQL ]    [ Prometheus (экспортер) ]
```

## 🚀 Как запустить
1. Клонируй проект
```bash
git clone https://github.com/timur-danilchenko/metric-bridge.git
cd metric-bridge
```
2. Установи зависимости
```bash
go mod tidy
```
3. Запусти воркер (тестовая версия)
```bash
go run ./cmd/worker
```

## 🗂️ Структура проекта
```plaintext
metric-bridge/
├── cmd/             # Точки входа для различных компонентов
│   └── worker/      # Основной воркер для обработки метрик
├── internal/        # Внутренние модули
│   ├── kafka/       # Работа с Kafka
│   ├── storage/     # PostgreSQL логика
│   ├── metrics/     # Prometheus метрики
│   └── processor/   # Бизнес-логика обработки метрик
├── configs/         # YAML-файлы конфигурации
├── go.mod
└── README.md
```
